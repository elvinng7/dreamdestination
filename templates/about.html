Welcome to the About Page!

We created an application to help users find their perfect vacation spot. In this application, there is a four question quiz that takes the users' preferences and uses a number mapping format to get the best spot based on the users' preferences. Each question on the quiz has a number value ranging from zero to four. The number values are then compared to numbers that are hardcoded into set locations. The locations that appear on the results page are hardcoded into the main.py file and then with the number mapping format, the locations that are already set have numbers that correspond to the the question values.

With this library, the website compares the users' preferences values with the values for each hardcoded location. Once all the numbers are compared, it finds the closest value to the users' preferences and prints out the result.
